# rubles_exchange_rate

<h1>Clone rep</h1>

<h2>backend start</h2>

<li>cd backend</li>
<li>python manage.py migrate</li>
<li>python manage.py set_admin_django</li>
<li>python manage.py runserver</li>

<h2>telegram bot start</h2>
<li>Run 'bot.py' (app.bot.py)</li>
